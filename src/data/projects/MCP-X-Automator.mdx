## Overview

**MCP Progress Poster** is a custom-built **MCP (Model Context Protocol) server** designed to automate _build in public_ updates for developers.

While consistently learning **Data Structures and Algorithms**, I wanted a frictionless way to share my daily progress publiclyâ€”without manually writing posts every day. This project solves that problem by converting real coding activity into meaningful social updates using AI-aware context.

## Motivation

The _build in public_ movement encourages developers to share what they learn daily. However, maintaining consistency becomes difficult when updates are repetitive and time-consuming.

While practicing DSA daily inside **VS Code**, I noticed that **GitHub Copilot** already understands:

- What I worked on
- Which problems I solved
- My learning focus for the day

Instead of manually summarizing progress, I decided to automate the entire workflow using an MCP server.

## How It Works

1. I practice DSA and write code throughout the day in VS Code
2. Copilot maintains contextual awareness of my work
3. I trigger a single command:

> **Post my todayâ€™s progress on Twitter**

4. The MCP server:
   - Extracts relevant learning context
   - Generates a concise and meaningful update
   - Publishes it directly to **Twitter** using the Twitter API
